# Roadmap

---

## Phase 1 · MVP（已完成）

- [x] Next.js 14 全栈骨架
- [x] 三页核心 UI（首页 / 详情页 / 生成页）
- [x] `lib/models.ts` 封装推理 / 文生图 / 语音合成三种调用
- [x] mock 模式：无 key 也能完整体验流水线
- [x] 3 集预制示例（DeepSeek-V3 / Mamba / Attention Is All You Need）

---

## Phase 2 · 接入真实 API

**目标**：把示例数据切换为实时生成的真实内容，开始周更。

- [ ] 配置 API key，关闭 mock 模式
- [ ] 跑通第一集真实播客
- [ ] 调优脚本 prompt：人设语气、知识密度、节奏
- [ ] 调优封面 prompt：风格统一性
- [ ] 调优 TTS：句尾停顿、专有名词发音
- [ ] 引入 SSE 流式生成，替代当前轮询占位
- [ ] 缓存层：相同论文 + 相同风格不重复消耗 token

---

## Phase 3 · 内容运营 + 用户自助

- [ ] **编辑部周更**：每周 5-7 集，覆盖 LLM / 视觉 / RL / 系统四大方向
- [ ] **用户自助生成**：登录后可粘贴任意 arXiv 链接生成专属播客
- [ ] **PDF 直传**：扩展到 NeurIPS / ICML / ICLR 等会议论文
- [ ] **Newsletter 同步**：每周精选邮件推送
- [ ] **多语言**：英文版 host / guest 配音
- [ ] **RSS 订阅**：接入苹果播客、小宇宙

---

## Phase 4 · 长期愿景

- [ ] **跨论文综述模式**：选 5 篇相关论文，AI 主播圆桌讨论
- [ ] **学习路径模式**：根据用户兴趣，自动编排"零基础到前沿"的播客系列
- [ ] **智能反馈**：用户对每集的"听完率""反复播放点"反向优化 prompt
